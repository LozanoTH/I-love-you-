## que es webGL
>  WebGL (Web Graphics Library) es una API de JavaScript que permite renderizar gráficos 2D y 3D interactivos directamente en navegadores web compatibles, sin necesidad de plugins, utilizando el hardware gráfico (GPU) de la computadora para una mayor velocidad y eficiencia. Esta tecnología es un estándar abierto mantenido por The Khronos Group, que se integra con elementos HTML y JavaScript para crear experiencias visuales sofisticadas y de alto rendimiento en la web. 
## Características clave
> Aceleración por Hardware: WebGL utiliza la unidad de procesamiento gráfico (GPU) del dispositivo, lo que permite renderizar gráficos más rápido y manejar visualizaciones complejas y juegos.
> 
## Sin Plugins:
> No requiere que los usuarios instalen software adicional, lo que lo hace accesible en cualquier navegador web moderno. 

## API de JavaScript: 
> Se controla mediante código JavaScript y se ejecuta en un elemento <canvas> de HTML. 

## Código Shader:
> Los programas WebGL incluyen código de control en JavaScript y código de efectos especiales en el lenguaje de sombreado (GLSL ES), que se ejecuta en la GPU.
> 
## Integración Web:

> Se integra con otras tecnologías web, como HTML y CSS, permitiendo mezclar elementos WebGL con otros contenidos de la página. 
Cómo funciona
## Contexto de JavaScript:

> El desarrollador utiliza JavaScript para crear y administrar un contexto de WebGL. 

## Lenguaje de Sombreado (GLSL):

> Se escriben programas en GLSL (OpenGL ES Shading Language) para definir cómo se dibujan los objetos y se aplican los efectos visuales en la GPU. 
## Buffers y Texturas:
> Se configuran búferes (buffers) para almacenar datos de vértices y texturas, que se envían a la GPU para ser procesados. 
## Renderizado en la GPU: 

> La GPU utiliza estos datos y shaders para renderizar los gráficos 2D o 3D en el elemento <canvas>. 
## Aplicaciones comunes
Visualizaciones de Datos: Para presentar grandes conjuntos de datos de forma interactiva. 
Juegos Web: Permite crear videojuegos 3D directamente en el navegador. 
Educación y Simulación: Proporciona experiencias de aprendizaje interactivas con modelos 3D. 
Cartografía Geoespacial: Se utiliza en aplicaciones para visualizar mapas y datos geoespaciales. 
Aplicaciones Empresariales: Para crear visualizaciones y herramientas interactivas en plataformas web.
